Local Voice/Audio Playback Disabled

FFmpeg is not installed or not configured in your system's PATH. The local music player cannot stream audio. Please install FFmpeg or invite one of the Alternative External Music Bots (like Lara) listed at the bottom of this tab.

📦 Step-by-Step FFmpeg Installation Guide

Windows Setup:
  1. Download the latest essentials build zip from Gyan.dev.
  2. Extract the folder and rename the main folder to ffmpeg.
  3. Move the ffmpeg folder directly into your C:\ drive (so it sits at C:\ffmpeg).
  4. Open Windows Search, search for "Environment Variables", and choose "Edit the system environment variables".
  5. Click "Environment Variables..." at the bottom.
  6. Under System variables, locate "Path", select it, and click "Edit...".
  7. Click "New", paste C:\ffmpeg\bin, click OK on all prompt screens.
  8. Restart the Bot launcher terminal to load the new environment path.
macOS (via Homebrew):

Open Terminal and run:
brew install ffmpeg

Linux (Ubuntu/Debian):

Open Terminal and install via package manager:
sudo apt update && sudo apt install -y ffmpeg
